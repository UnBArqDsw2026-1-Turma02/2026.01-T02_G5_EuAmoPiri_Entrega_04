# Módulo 13 — Reações por Emoji (Frontend)

Documento de reutilização de software para **RF13** (usuário reage a um relato com emoji) no **frontend** Eu Amo Piri. Persistência das reações está em [backend/04.ComentariosReacoes.md](/ArquiteturaReutilizacao/backend/04.ComentariosReacoes.md).

---

## 1. O que foi implementado

| RF | Página / componente | Integração |
|----|---------------------|------------|
| RF13 | Botões ❤️ Amei · 👍 Gostei · 👎 Não gostei em cada `CommentCard` | `experienceAdaptor.reactToExperience` → `POST /places/:id/experiences/:expId/reactions` |
| RF13 | Atualização otimista de contador — sem aguardar servidor | `Map<expId, emojiKey>` em `userReactions`; 1 reação por comentário |
| RF13 | Troca/desfazer reação | Clicar no mesmo emoji desfaz; clicar em outro troca (remove anterior, adiciona novo) |

---

## 2. Bibliotecas e padrões reutilizados

| Componente | Origem | Papel |
| --- | --- | --- |
| **CommentCard** (inline em `PlaceDetailPage`) | Implementação própria | Componente existente estendido com botões de reação sem criar novo arquivo |
| **experienceAdaptor.reactToExperience** | *Adapter* | Registra reação na API em background; fallback mock silencioso (estado já atualizado otimisticamente) |
| **useState** — `userReactions` (`Map`) | React | Rastreia reação ativa por comentário; atualização imutável via `new Map(prev)` |
| **Optimistic UI Pattern** | Implementação própria | Contador atualizado imediatamente no cliente; servidor confirmado em background |

---

## 3. Rastreabilidade

| Critério | Evidência frontend |
|----------|--------------------|
| RF13 — usuário reage com emoji a um relato | Botões em `CommentCard` com `aria-label` (ex.: `"Amei: 3"`) |
| RF13 — 1 reação por comentário, anulável | `userReactions.get(expId)` === emoji atual → remove; diferente → troca |
| RF13 — contador reflete estado real | Atualização otimista em `setExperiences`; chamada ao adaptor em background |
| RF13 — reação ativa destacada visualmente | `.reactionBtnActive` com fundo `#f0f4e8` e borda `#6B8E23` |

Backend correspondente: [04.ComentariosReacoes](/ArquiteturaReutilizacao/backend/04.ComentariosReacoes.md).

Documento detalhado: [rf13-emojis](/docs/rf13-emojis.md).
